# Tablas

App para practicar las tablas de multiplicar del 1 al 12.

## Características
- Practicar tablas específicas o mezcladas
- Desafío cronometrado de 60 segundos
- Mapa de progreso con colores según velocidad
- Soporte para múltiples usuarios

## Uso
Simplemente abre `index.html` en un navegador.

## Tecnologías
- HTML, CSS, JavaScript vanilla
- Sin dependencias externas
